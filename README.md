# PavanXDCL Landing Page 🚀

A modern educational landing page built using HTML and CSS for the PavanXDCL community. The website provides access to DSA learning resources, aptitude preparation, coding mentorship, and placement-focused content.

## 📌 Project Overview

PavanXDCL is a learning platform designed to help students prepare for:

- Data Structures & Algorithms (DSA)
- LeetCode Practice
- Aptitude Preparation
- Placement Interviews
- Full Stack Development

The project consists of:

- Landing Page (`Pavan.html`)
- DSA Pro Page (`pro.html`)
- Responsive UI using HTML & CSS

---

## ✨ Features

### Home Page
- Modern navigation bar
- Community links
- DSA & Aptitude learning resources
- Statistics section
- Community engagement section

### DSA Pro Page
- DSA roadmap
- 500+ hours learning content showcase
